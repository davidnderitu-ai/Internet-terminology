Extranet

An Extranet is a controlled private network that allows access to partners, vendors, suppliers, or an authorized set of customers. It is essentially an Intranet that provides a "window" for outsiders to collaborate.

Key Features:

  • Allows external collaboration with trusted third parties.
  • Requires authentication (login credentials) for access.
  • Facilitates supply chain management and B2B communication.
