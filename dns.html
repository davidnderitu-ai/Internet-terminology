Domain Name System (DNS)

DNS is a hierarchical and decentralized naming system for computers, services, or other resources connected to the Internet. It translates human-readable domain names (like google.com) into the numerical IP addresses (like 142.250.190.46) needed to locate and identify computer services and devices.

How DNS Works:

  1. Request: You type a URL into your browser.
  2. Lookup: Your computer asks a DNS recursive resolver for the IP address.
  3. Resolution: The resolver finds the IP address from the DNS nameservers.
  4. Connection: Your browser uses the IP address to connect to the web server.
