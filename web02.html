Web 2.0: The Social Web

Web 2.0 refers to the second generation of the World Wide Web, which shifted from static pages to dynamic, user-generated content. This era saw the rise of social media, blogs, and video-sharing platforms.

Key Characteristics:

  • User Participation: Users create content through comments, reviews, and social media posts.
  • Rich User Experience: Websites became interactive and responsive (using technologies like AJAX).
  • Social Connectivity: Platforms like Facebook, YouTube, and X (Twitter) connect people globally.
