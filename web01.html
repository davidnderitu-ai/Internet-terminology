Web 1.0: The Read-Only Web

Web 1.0 refers to the first stage of the World Wide Web's evolution, roughly from 1991 to 2004. It was characterized by static websites that provided information but allowed for very little user interaction.

Characteristics of Web 1.0:

  • Static Pages: Content did not change unless the webmaster manually updated the files.
  • Read-Only: Users were consumers of information, not creators. No social media or comments.
  • Basic Design: Use of frames, tables, and simple HTML tags with very little CSS.
