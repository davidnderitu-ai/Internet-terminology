# Internet Terminology school Project

This repository is a comprehensive web-based guide to fundamental internet concepts and terminology. It was developed as part of a BBIT assignment at Strathmore University.

## Project Overview
The goal of this project was to create a functional, minimalist website that explains core web technologies and the evolution of the internet from Web 1.0 to Web 4.0.

## Key Concepts Covered
- **Core Protocols:** TCP/IP, HTTP, FTP, and DNS.
- **Web Infrastructure:** Web Servers, Browsers, and URLs.
- **Architecture:** 3-Tier/Multitier Architecture.
- **Network Types:** Intranets and Extranets.
- **Evolution:** The progression from static (Web 1.0) to intelligent/symbiotic (Web 4.0) environments.

## Technical Implementation
- **HTML5:** Used for semantic structure across 17 distinct pages.
- **CSS3:** Applied a functional minimalist design focusing on readability and clean layout.
- **Version Control:** Managed via Git/GitHub with a focus on incremental commits and professional documentation.
